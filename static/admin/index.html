클라이언트 ID 만드는 법 (1회)
1. Google Cloud 콘솔 → 사용자 인증 정보
2. 사용자 인증 정보 만들기 → OAuth 클라이언트 ID → 웹 애플리케이션
3. 승인된 자바스크립트 원본에 추가: https://seunghoonchoi.com · http://localhost:1319
4. 만들고 나온 클라이언트 ID(...apps.googleusercontent.com)를 위에 붙여넣기.
※ OAuth 동의 화면이 "테스트" 상태면, 테스트 사용자에 본인 이메일을 추가하세요.
