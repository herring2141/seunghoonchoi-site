GitHub 토큰

토큰 만드는 법(1회): GitHub → Settings → Developer settings → Fine-grained tokens → Generate new token → Repository access = seunghoonchoi-phd/seunghoonchoi-site만 → Permissions의 Contents를 Read and write로 → 만료일(예: 90일) 설정 후 생성 → 토큰을 복사해 위에 붙여넣기.

토큰은 이 브라우저에만 저장되고 서버로 보내지 않습니다. 이 저장소의 파일만 다룰 수 있고, GitHub에서 언제든 폐기할 수 있습니다.
